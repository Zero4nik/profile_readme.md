## 🏆 Ключевые компетенции

**Архитектура:** проектирую многослойную структуру (Routes → Controllers → Services → Repositories), разделяю ответственность, пишу чистый код без `as any`.

**Backend:** JWT-аутентификация, CRUD с Prisma ORM, cursor-based пагинация, кэширование Redis с TTL и инвалидацией, WebSocket через Socket.IO, CORS, Helmet, Rate-limit.

**Frontend:** Next.js App Router, React 19, защищённые роуты, кастомные хуки, оптимистичное обновление UI с откатом при ошибках, Axios с интерсепторами, адаптивная вёрстка.

**DevOps:** Docker Compose (PostgreSQL + Redis), GitHub Actions (автопрогон тестов, проверка типов, миграции), деплой на Vercel (фронт) и Render (бэк).

**Тестирование:** Jest, Supertest, Vitest, Testing Library, Postman-коллекции с переменными окружения.
